Implementing Linear Regression Model¶

Okay, so we need to define a dependent and independent variable. One of these isn't clearly dependent on the other, so it's a bit of a misnomer. But, a common argument in basketball circles is that PPG is a bad metric for judging how good a player is. PER, while very flawed, is meant to do just that.

We have seen PER correlate with PPG somewhat linearly. So, we can use a linear regression model, give it a PPG, and try to predict a player's PER.

Defining our linear regression model¶

We want to use f(x) = wx + b, with w and b initially chosen at 0.

We also need to define m, the # of data points, and gamma, the learning rate, which is the scaling factor for the derivative of the error function.

We'll define the "dependent" variable, PPG, as x, and the "independent" variable, PER, as y.

Lastly, we make an array for the model prediction, fx

So we have:

w, b, m, gamma, x, y, f(x)

#This function will fit a line to the data given w, b, and store that in fx
def linreg(x,m,w,b):
    for i in np.arange(0,m,1):
        fx[i] = w * x[i] + b
        
    return fx

Finding the ideal w, b¶

We use the gradient descent formulation and a while loop to minimize the error function and find the best w, b

We'll go with an accuracy of 0.001, and an iteration limit of 10000

In [11]:
def ideal_w_b(fx, m, y, gamma, w, b, wnew, bnew):
    
    for i in range(100000):
        sum_of_fx_minus_y = 0
        sum_of_fx_minus_y_times_x = 0
        for i in np.arange(0,m,1):
            sum_of_fx_minus_y_times_x =  sum_of_fx_minus_y_times_x + (fx[i] - y[i])*x[i]
            sum_of_fx_minus_y =  sum_of_fx_minus_y + (fx[i] - y[i])
            
        wnew = w - (gamma*sum_of_fx_minus_y_times_x)/m
        bnew = b - (gamma*sum_of_fx_minus_y)/m
        
        #implement linreg again
        fx = linreg(x,m,wnew,bnew)
        
        if (abs(wnew - w) < 0.00001 and abs(bnew-b) < 0.00001):
            w = wnew
            b = bnew
            return fx, w, b
            break
            
        w = wnew
        b = bnew
        if i == 99999:
            print('Failed to converge')
            return w, b

Now that we have our functions, let's run our code!
